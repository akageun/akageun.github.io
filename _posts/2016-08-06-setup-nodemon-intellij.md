#1. nodemon install
 - 멍령어 : sudo npm install -g nodemon

# 2. 설정 path 확인
- 명령어 :  which nodemon
- 결과 :  /usr/local/bin/nodemon

# 3. Intelli J 내 세팅
- Run/Debug Configurations -> Node parameters -> /usr/local/bin/nodemon
