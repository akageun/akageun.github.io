## undef
- 초기화하지 않은 변수는 `undef`
```perl
my $test;

if (defined($test)) {
    print("Undefined");
} else {
    print("Exist");
}
```

## hash
```perl
my %hash = ();

my %hash = (
    "a" => "a_1", 
    "b" => "b_1", 
    "c" => "c_1", 
    "d" => "d_1"
);
```

#### delete key
```perl
my $delete_target_key = "a";
delete %hash{$delete_target_key};

```

#### get keys
- hash 의 key 값 가져오기

```perl
my @keys = keys %hash;
```

#### get values
- hash 의 value 값 가져오기

```perl
my @values = values %hash;
```
